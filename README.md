# Iris Flower Classifier

This project uses the classic Iris dataset to train a machine learning model that predicts flower species based on petal and sepal measurements.

## Features
- Uses scikit-learn
- RandomForestClassifier model
- 95%+ accuracy
- Clean and simple notebook
